# Website Kelas Ilmu Komputer 23 B Universitas Negeri Medan

## Tujuan

Website ilmu komputer 23 b ini bertujuan untuk menampilkan struktur kelas yang terdiri dari ketua kelas atau komisaris tingkat, sekretaris dan juga bendahara. Pada website ini juga menampilkan galeri dan juga jadwal perkuliahan yang mempermudah mahasiswa.

## Fitur

- Team Generator
- Spin The Wheel
- Hitung Umur
- Game Slot
- Flappy Bird
- Tic Tac Toe
- Helium AI

## Teknologi yang digunakan

- Html
- Css
- Javascript
- Google analytics
- Scroll Reveal
- SwiperJS
- Formspree
- SweetAlert2
- API Gemini 1.5
